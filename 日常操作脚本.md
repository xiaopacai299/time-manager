一套可直接用的“彻底清理 + 重装”命令（Windows）
在仓库根目录执行：

# 1) 停掉可能占用 node_modules 的进程
taskkill /F /IM node.exe

# 2) 删除所有 node_modules（根 + 各 workspace 包）
cmd /c "rmdir /S /Q node_modules"
cmd /c "rmdir /S /Q packages\desktop\node_modules"
cmd /c "rmdir /S /Q packages\mobile\node_modules"
cmd /c "rmdir /S /Q packages\server\node_modules"
cmd /c "rmdir /S /Q packages\shared\node_modules"

# 3) 可选：删 pnpm 缓存（一般不需要，排查疑难再用）
# pnpm store prune

# 4) 重新安装
pnpm install
